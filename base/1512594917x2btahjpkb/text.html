Инфо, касающаяся запуска подсистем, хранится в HKLM\SYSTEM\CurrentControlSet\Control\Session Manager\SubSystem


В Required перечисляются подсистемы, загружаемые при загрузке

системы. Значение содержит две строки: Windows и Debug.

Значение Windows содержит спецификацию файла подсистемы Windows, Csrss.exe (от Client/Server Run-Time Subsystem, клиент-серверная подсистема времени выполнения).

Значение Debug оставлено пустым (поскольку оно используется для внутреннего тестирования) и поэтому не вызывает никаких действий.

Значение Optional показывает, что по запросу будет запущена та или иная подсистема.

Значение реестра Kmode содержит имя файла той части подсистемы Windows, которая работает в режиме ядра, Win32k.sys.
